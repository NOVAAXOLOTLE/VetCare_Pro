# VetCare Pro

Android app built with Kotlin, Jetpack Compose Material 3, Clean Architecture, Hilt, Navigation Compose, and Firebase-ready data layers.

## Run

```bash
./gradlew assembleDebug
```

## Notes

- Demo login accounts are seeded in the local repository.
- Firebase, Firestore, Storage, Messaging, Maps, and ZXing dependencies are wired in the project.
- Offline veterinary guides load from `app/src/main/assets/offline_guides.json`.
